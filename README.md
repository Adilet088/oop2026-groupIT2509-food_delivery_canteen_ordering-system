# Food Delivery / Canteen Ordering System

## Project topic
Ordering system for a canteen or delivery service with menu management and order tracking.

## Course
Object Oriented Programming (Java)

## Implemented features (Milestone 1)
- Entities: Customer, MenuItem, Order, OrderItem
- Database: PostgreSQL (Supabase)
- Tables: customers, menu_items, orders, order_items
- JDBC connection via DatabaseConnection
- Repositories and services separation
- Exception handling (invalid quantity, not found, unavailable menu item)
- Basic user flows:
  - View available menu
  - Place an order
  - View active orders
  - Complete order
